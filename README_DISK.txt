SkyEditor是一款方便易用的文本编辑器。通过一个开箱即用的方式
方式提供用户一个良好的编辑使用体验，让用户专注于自己有创意
的学习和工作。

版本信息：SkyEditor 标准版 1.0.0

支持的平台：
1. Windows 10, 11.
2. Apple macOS - M1, M2等系列。
